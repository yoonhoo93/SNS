## 🚀 프로젝트 초기 세팅

### 1. 프로젝트 생성

```bash
npm create vite@6.5.0 프로젝트명
```

> `vite@6.5.0` : Vite 버전을 6.5.0으로 지정해서 프로젝트를 생성

---

### 2. TailwindCSS 설치

```bash
npm install tailwindcss @tailwindcss/vite
```

---

### 3. TailwindCSS 적용

TailwindCSS 설치 후 전역 CSS 파일 (`src/index.css`) 에 아래 코드를 추가한다.

```css
@import "tailwindcss";
```

---

## 📁 기본 실행

```bash
npm install
npm run dev
```
